1. **Clone the Repository**

   cd your-repo

2. **Install Dependencies**

   - For the backend:
     
     cd server  
     npm install

   - For the frontend:
     
     cd client  
     npm install

3. **Start the Development Servers**

   - **Server**: Run the following to start the Node.js server:
     
     npm start

   - **Client**: Run the following to start the Angular development server:
     
     npm run start
